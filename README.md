# Gmail-filters
（我常用的）繁體中文 Gmail 篩選器規則。  
不一定能涵蓋到所有人的使用情境，有些規則也可能有重複或遺漏。
## 語法
`{}` 為 OR，`()` 為 AND，`subject:` 代表以電子郵件標題進行篩選，更詳細的語法可以參考 [在 Gmail 中修正搜尋結果](https://support.google.com/mail/answer/7190?hl=zh-Hant)。
